OpenBDC is a BDM pod for MCU's that use Freescale's single wire debug interface. The OpenBDC firmware is based on the S12CForth project allowing the pod to be used as simple protocoll translater, as well as high level debugger. The project started out with it's own hardware, a low-cost MC9S12C128 based debug interface (the original OpenBDC pod).

Upcoming releases will also support Mini-BDM-Pod and LFBDMPGMR devices.
